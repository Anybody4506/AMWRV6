# AMWRV6

Lua scripts for **Aimware V6**.

This repository is a place where I share various Lua scripts made for Aimware V6.  
They are mainly small tools and helpers that extend or tweak the existing features.

The community is welcome to improve any of the scripts:
- suggest ideas or changes
- report issues
- open pull requests
- share configs or data files related to the scripts

Everything here is open to feedback and contributions.
